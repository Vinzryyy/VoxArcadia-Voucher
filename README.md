# VoxArcadia Voucher

Company Profile Web-Based project.

## Description
VoxArcadia is a fictional digital game voucher platform that provides fast,
secure, and modern top-up services for online games.

## Features
- Responsive UI (Bootstrap 5)
- Smooth scrolling navigation
- Auto price calculation
- Login & Register simulation
- Top-up form simulation

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Deployment
- GitHub Pages
